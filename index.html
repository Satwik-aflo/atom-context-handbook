Update Rules

When Required action Why
A new public HTML artifact is created Add a card to this hub, publish the artifact under its own route, then republish this root index.html. The root page should remain the single doorway to all generated docs.
An existing public HTML artifact is renamed or moved Update the card link, source link, title, and route in this hub. Old links should not quietly go stale.
A local-only HTML experiment should not be public Do not add it here unless the user explicitly approves public publishing. The Pages repo is public.
